## 背景

| 任务名称 | 执行时间 | 内容 | 备注 |
|---------|---------|------|------|
| `cmcc-daily-8am` | `0 8 * * *` | 移动每日完整日报推送 | 保持不变 |

## 用户偏好
1. **夜间不执行监控**：18:00-次日9:00期间不运行任何监控任务
2. **工作时间监控**：9:00-18:00每小时运行一次移动小时监控
3. **简洁通知**：小时监控只通知有无新公告，不发送详细内容

## 执行时间线
```
07:00-08:00 不运行
08:00 移动日报推送
08:15 电信日报推送
09:00-18:00 移动小时监控（每小时）
18:01-次日08:59 不运行
```

## 配置命令
```bash
# 查看当前定时任务
hercules cronjob --list

# 调整移动小时监控任务
hercules cronjob --name cmcc-hourly-monitor --schedule "0 9-18 * * *"

# 验证时间设置
echo "当前移动小时监控任务时间：$(hercules cronjob --name cmcc-hourly-monitor | grep -o 'schedule:.*')"
```
